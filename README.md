# ✦ Aether Weather

A royal, elegant weather app with live data from OpenWeatherMap. Clean dark theme with gold accents, smooth animations, and a premium feel.

## 🌤️ Features

- 🔍 **City search** — search any city in the world
- 📍 **Auto-location** — detect your location with one click
- ⏱ **Recent searches** — saved automatically, click to reload
- 🌡 **Current weather** — temperature, feels like, humidity, wind, visibility, pressure, cloud cover
- 🌅 **Sunrise & Sunset** times
- ⏰ **Hourly forecast** — next 36 hours (scrollable)
- 📅 **5-Day forecast** — daily high/low with weather icons
- 💎 **Royal dark theme** — deep navy, gold, plum — glassmorphism cards
- 📱 **Fully responsive** — mobile & desktop

## 🔑 API

Uses [OpenWeatherMap](https://openweathermap.org/api) free tier:
- Current Weather API
- 5-Day / 3-Hour Forecast API
- Geocoding API (city name → coordinates)

## 🛠️ Deploy to GitHub Pages

1. Create repo `weather-app` on GitHub
2. Upload all 3 files
3. Settings → Pages → main branch → Save
4. Live at: `https://YOUR_USERNAME.github.io/weather-app`

## 🧠 What This Project Teaches

- Fetching data from a real REST API using `fetch()`
- Working with `async/await` and `Promise.all()`
- The OpenWeatherMap API structure
- Browser Geolocation API
- `localStorage` for persistent recent searches
- Glassmorphism CSS design
- Responsive layouts
